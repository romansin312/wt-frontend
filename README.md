## Getting Started

### Installation

Install the dependencies:

```bash
npm install
```

### Config

Set the backend base URL in the 'config.ts' file.

### Development

Start the development server with HMR:

```bash
npm run dev
```

Your application will be available at `http://localhost:5173`.
